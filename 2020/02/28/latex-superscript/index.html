LaTeX的公式上标（下标）垂直高度调整

发表于 2020-02-28 更新于 2020-02-29 分类于 LaTeX

如下图所示，在使用LaTeX\LaTeXLATE​X写作时，发现对于公式中同一个对象，只有上标 t(j)~t^{(j)} t(j)（或只有下标 ti~t_i ti​）时与上下标同时存在 ti(j) ~t^{(j)}_i~ ti(j)​ 时，两种情况下上标（下标）的垂直高度位置并不一致，即Equ1。

实际上，这体现了LaTeX\LaTeXLATE​X排版的内容与样式分离的特点，即可以根据内容动态调整。

但是，几个公式出现在一起，上下标位置不一致，看上去就很难受Orz

有两种解决方法：

  1. 使用宏包xcolor，令上下标同时存在，将不需要显示的上标（下标）设置为white

    这种方法还是将上标（下标）嵌入到PDF文档中，是实际存在的（可以选中复制）

  2. 使用命令substack，手动微调

    这种方法没有将上标（下标）嵌入到PDF文档中
