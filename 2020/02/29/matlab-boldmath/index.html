MATLAB中插入LaTeX粗斜体数学公式

发表于 2020-02-29 更新于 2020-03-20 分类于 LaTeX

MATLAB可通过定义Interpreter类型，来选择文本字符解释器：

  1. none - 显示字面字符（默认）
  2. tex - 使用TeX\TeXTE​X标记子集解释字符
  3. latex - 使用LaTeX\LaTeXLATE​X标记解释字符

在线帮助文档给出了MATLAB支持的LaTeX\LaTeXLATE​X命令，包括用于生成粗正体数学公式的\bf命令

例如，y=2x\bf{y=2x}y=2x \bf{y=2x}

而不包括LaTeX\LaTeXLATE​X的amsmath宏包中常用的\bm或\boldsymbol命令，也就无法使用这些命令

粗斜体数学公式用于表示矢量x\boldsymbol{x}x、v\boldsymbol{v}v、a\boldsymbol{a}a，矩阵A\boldsymbol{A}A、B\boldsymbol{B}B、C\boldsymbol{C}C等
