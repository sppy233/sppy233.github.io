二刺螈：Hexo添加Live2D看板娘

发表于 2020-03-02 更新于 2020-03-05 分类于 Hexo

二刺螈看板娘见GitHub项目hexo-helper-live2d，现实效果见我的blog界面右下角。

我不是二刺螈，但是搁这儿整只猫，也害行。

模型效果：动态展示、静态展示

需要注意，若直接将模型上传GitHub，加载到blog属实很慢。

建议使用CDN加速，jsDelivr是一个免费的开源CDN平台，平台上已经缓存了所有的模型，下面以这只猫（模型名称为live2d-widget-model-hijiki）为例，介绍设置方式：

  1. 平台主界面检索模型名称，点击进入

  2. 在npm CDN Files中找到文件夹/npm/live2d-widget-model-hijiki@1.0.5/assets，点击进入

  3. 找到文件/npm/live2d-widget-model-hijiki@1.0.5/assets/hijiki.model.json，选择Copy URL

  4. 将复制的CDN链接粘贴到live2d.model.use下，即

仅研究发现，jsDelivr平台上看板娘模型的CDN链接一般是https://cdn.jsdelivr.net/npm/live2d-widget-model-NAME@1.0.5/assets/NAME.model.json，仅需将CDN链接中的NAME改为具体模型名称即可。

若hexo s或hexo d后没有显示看板娘，可能是CDN链接出错，建议还是按上面的步骤设置

jsDelivr平台上有每个模型的访问次数：

模型名称 访问次数 说明
hijiki 黑猫
tororo 白猫，与NexT主题颜色不搭，看不清
wanko 狗子
shizuku 默认选项，二刺螈最爱
hibiki 我觉得害行
haru 有两个模型，我觉得haru01害行

点击访问次数，也可连接到jsDelivr平台上相应的模型文件
