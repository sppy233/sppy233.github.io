NumPy的Array结构

  1. Array可以是多维列阵（包括矩阵），且按行主顺序存储的，而MATLAB则是按列主顺序存储；

  2. 三维Array的形状是(D, R, C)，等价于MATLAB中三维矩阵(R, C, D)，例如

    R: row, C: column, D: depth
